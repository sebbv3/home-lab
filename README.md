# IT Home Labs

This repository contains hands-on IT labs focused on Linux, networking, troubleshooting, Docker, Nginx, and Helpdesk support scenarios.

## Goal

The goal of this project is to build practical skills for entry-level IT Support, Helpdesk, Networking, and Cloud Support roles.

## Labs Included
### 1. Networking Troubleshooting
Practiced basic network checks using common troubleshooting commands.

Skills:
- IP address checking
- DNS testing
- Ping tests
- Gateway understanding
- Internet connectivity troubleshooting

### 2. Nginx Web Server
Installed and managed an Nginx web server on Linux.

Skills:
- Web server basics
- systemctl
- Service status checks
- Localhost testing

### 3. Docker Nginx Lab
Ran an Nginx web server inside a Docker container.

Skills:
- Docker images
- Docker containers
- Port mapping
- Container start/stop
- Basic container troubleshooting

### 4. Helpdesk Scenarios
Practiced real-world Helpdesk tickets and troubleshooting flows.

Scenarios:
- User cannot connect to VPN
- Account locked
- Outlook not receiving emails
- Printer offline
- Internet/DNS issue

### 5. Scripts
Created simple scripts for IT troubleshooting and automation.

Skills:
- Bash scripting
- Network diagnostics
- Basic automation
